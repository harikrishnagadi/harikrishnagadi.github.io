01 Demos
Demo 1 · Driver Fatigue Monitoring
Real-time DMS with Proactive Safety Interventions
Live driver monitoring using MediaPipe face analysis — tracks eye closure (PERCLOS), yawning (MAR), blink rate, and head pose. When drowsiness index crosses thresholds, the system proactively intervenes: adjusting cabin temperature, suggesting rest stops with navigation, or triggering safety alerts through the infotainment system.
Computer Vision MediaPipe PERCLOS Fatigue Detection Real-time
Demo 2 · Proactive Caller Assistant
Context-Aware AI Agent for Phone Calls
Context-aware AI agent that proactively assists during phone calls with relevant information, calendar context, and smart suggestions across devices. The system anticipates what the driver needs during a call and surfaces relevant data without being asked.

02 Capabilities
📷
Driver Monitoring (DMS)
Real-time face analysis: EAR, MAR, PERCLOS, blink rate, head pose, yawn detection. Five-level drowsiness classification from Alert to Critical.
🧠
LLM Deliberation
Context-aware reasoning engine that weighs driver state, vehicle conditions, time of day, and route context to decide when and how to intervene.
🌍
Navigation Integration
Automatic rest stop discovery and route suggestion when fatigue is detected. Leaflet-based map visualization with real-time route overlay.
🌡
Cabin Climate Control
Proactive HVAC adjustment — lowers cabin temperature to increase alertness during early drowsiness stages before escalating to alerts.
📱
Smart Cockpit Companion
Native integration with the vehicle OS — calendar, contacts, call state. Enables context-aware assistance beyond standard infotainment capabilities.
🛡
Safety-First Design
No covert interventions. Intervention cooldown (30s) prevents alert fatigue. Tier 1 signals only for triggers. GDPR + China PIPL compliant.
